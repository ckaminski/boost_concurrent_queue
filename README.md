### Goal

A Boost-based multi-threaded concurrent queue that has notify support, and allows thread-safe producer-consumer patterns.

### Prereqs 

* CMake 3.0
* Boost 1.50

### Building

1. git clone <this repo> 
2. git submodule init
3. git submodule update
4. mkdir build
5. cmake .. 
6. make test
